# Social / source icons (Instagram, YouTube, TikTok)

To use **real brand icons** instead of SF Symbols:

1. Download PNGs from Flaticon (free with attribution if required):
   - **YouTube**: https://www.flaticon.com/free-icon/youtube_1384060  
   - **Instagram**: https://www.flaticon.com/free-icon/instagram_15713420  
   - **TikTok**: https://www.flaticon.com/free-icon/social-media_15713399  

2. Add each file to its imageset in this folder:
   - Save as `youtube.png` → put in `YouTubeIcon.imageset/`
   - Save as `instagram.png` → put in `InstagramIcon.imageset/`
   - Save as `tiktok.png` → put in `TikTokIcon.imageset/`

3. Use 512×512 or 1024×1024 PNG for best quality; the app scales them.

Until you add these files, the app shows SF Symbol fallbacks (play.rectangle.fill, camera.fill, music.note).
